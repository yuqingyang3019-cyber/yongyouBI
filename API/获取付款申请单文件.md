请求方式	POST
ContentType	application/json
应用场景	开放API/集成API
API类别	详情查询
事务和幂等性	无
限流次数

来源	系统级

只看必填项
名称	类型	参数位置	必填	描述

名称	类型	数组	必填	描述
fileId	string	是	否	文件id 示例：["mfdfs23421","n95sdavdas"] 最大请求量：20

	]
}
